\section{Discussion}\label{sec:discussion}

\subsection{Template Evaluation}

The Seka template successfully meets all stated objectives. Unlike the rho-class template, which required specific fonts and Python dependencies, this template compiles reliably on any standard \LaTeX\ installation.

\subsection{Advantages}

Key advantages of this template include:

\begin{itemize}
    \item \textbf{No external dependencies:} Works with standard \LaTeX\ distributions
    \item \textbf{Modular structure:} Easy collaboration on large documents
    \item \textbf{Professional output:} Meets academic publication standards
    \item \textbf{Customizable:} Easy to modify colors, fonts, and layout
\end{itemize}

\subsection{Limitations}

Current limitations include:

\begin{itemize}
    \item Limited to two-column layout (one-column not supported)
    \item Basic code highlighting compared to specialized packages
    \item Requires manual appendix management
\end{itemize}

\subsection{Future Improvements}

Planned enhancements include \cite{futurework2025}:

\begin{itemize}
    \item One-column variant for specific document types
    \item Enhanced code highlighting with minted (optional)
    \item Automated glossary and acronym management
    \item Supplementary material handling
\end{itemize}

\begin{important}
Users are encouraged to modify the \texttt{Seka.cls} file to add features specific to their discipline.
\end{important}
